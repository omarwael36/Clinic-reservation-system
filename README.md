# Clinic Reservation System

A full-stack clinic reservation system built with Angular, Go, and MySQL.

## Features
- Doctor and Patient authentication
- Appointment booking system
- Doctor schedule management
- Real-time slot availability

## CI/CD Pipeline
This project uses Jenkins for automated Docker image building and deployment to DockerHub.

## Last Updated
Time: 7:30 PM 12/1/2023

### Technical Stack
- Frontend: Angular 16 with nginx
- Backend: Go with Gin framework
- Database: MySQL 8.0
- Containerization: Docker
- CI/CD: Jenkins
